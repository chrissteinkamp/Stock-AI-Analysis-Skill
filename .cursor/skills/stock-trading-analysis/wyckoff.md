# Wyckoff Method — Distribution, Accumulation, and Trading

Richard Wyckoff's framework maps **institutional (composite operator) behavior** through price/volume structures. Use it alongside candlesticks and MAs for **topping/bottoming** context.

**Reference:** [TrendSpider Wyckoff Distribution](https://trendspider.com/learning-center/chart-patterns-wyckoff-distribution/), [BitMEX Wyckoff guide](https://www.bitmex.com/blog/wyckoff-distribution), [Chart Guys](https://www.chartguys.com/chart-patterns/wyckoff-distribution).

---

## Core laws (apply to targets)

| Law | Application |
|-----|-------------|
| **Supply/Demand** | Price rises on demand dominance; falls on supply |
| **Cause & Effect** | Time spent in range (cause) projects magnitude of move (effect) |
| **Effort vs Result** | Volume (effort) must match price move (result); divergence = warning |

**Cause–Effect target:** Measure vertical height of trading range → project equal distance from breakout/breakdown point.

---

## Wyckoff Distribution (bearish — short framework)

Forms after **sustained uptrend**. Smart money sells into retail demand inside a range before markdown.

**Historical resolution:** Genuine distribution schematics resolve **downward ~65–70%** of the time when correctly identified ([Chart Guys](https://www.chartguys.com/chart-patterns/wyckoff-distribution)). Misidentified ranges in strong bull markets fail often — phase identification is critical.

### Phases A–E

| Phase | Name | What happens | Key events |
|-------|------|--------------|------------|
| **A** | Stopping action | Uptrend halts | **PSY** (preliminary supply), **BC** (buying climax — high volume top), **AR** (automatic reaction — sharp drop), **ST** (secondary test of BC on lighter volume) |
| **B** | Building cause | Range-building; institutions distribute | Price oscillates; volume often **lighter on rallies, heavier on declines**; **UT** (upthrusts) trap buyers |
| **C** | Test / trap | Final demand test | **UTAD** (upthrust after distribution) — false breakout above range; traps late longs. **Mirror of Spring in accumulation** |
| **D** | Weakness within range | Supply dominates | **SOW** (sign of weakness) breaks support; **LPSY** (last point of supply) = weak rally, lower high — **prime short entry zone** |
| **E** | Markdown | Trend down outside range | Breakdown accelerates; former support = resistance |

### Volume signature (distribution)

- **BC / UTAD:** High volume
- **Rallies inside range:** Declining volume (supply absorbing demand)
- **SOW / breakdown:** Expanding volume
- **LPSY rallies:** Light volume

### Short entry styles (Wyckoff)

| Style | Entry | Stop | Est. success if phase correct |
|-------|-------|------|------------------------------|
| **Aggressive** | Fade **UTAD** (failed breakout above range) | Above UTAD high | **~55–60%** — early, more false signals |
| **Standard** | **SOW** breakdown below range support | Back inside range / above LPSY | **~60–65%** |
| **Conservative** | **LPSY** retest (weak bounce) | Above LPSY / range midpoint | **~65–70%** — best R:R, may miss fast breakdowns |
| **Confirmation** | Phase **E** breakdown retest | Above range floor | **~60%** — lower reward, higher clarity |

### Bitcoin Nov 2021 — canonical distribution example

User reference case; use when parabolic asset shows topping range:

| Event | BTC ~2021 mapping |
|-------|-------------------|
| Markup | ~$10K → **$69K** parabolic rally |
| **BC** | Nov 2021 ATH **~$69,000** on euphoria + high volume |
| **AR** | Drop to **~$42K** (May–Jul 2021 consolidation floor) |
| Range / Phase B | **$42K–$69K** multi-month range; repeated failed highs |
| **UTAD** | Nov 10 ATH marginal new high, immediate rejection — classic trap |
| **SOW** | Break below **~$42K** support (Dec 2021 – Jan 2022) |
| **Phase E** | Markdown to **~$17K** (2022 bear) |

**Lesson for SNDK-style parabolic stocks:** Extended markup → range that looks like "consolidation" → UTAD at highs → SOW breakdown. FOMO buyers trapped at top. Tactical short aligns with **Phase C/D/E**, not Phase A.

### Distribution probability boosts (for output)

| Identified phase | Short bias confidence | Target markdown probability |
|------------------|----------------------|----------------------------|
| Phase A only (early topping) | Low–medium | **~40–50%** near-term decline |
| Phase B (range, no UTAD yet) | Medium | **~50–55%** — wait for C/D |
| Phase C (UTAD rejected) | High | **~60–70%** |
| Phase D (SOW + LPSY) | High | **~65–75%** to measured move target |
| Phase E (markdown) | Confirmation | **~60%** continuation to cause–effect target |

## Wyckoff Accumulation (bullish — long framework)

Mirror of distribution at **market bottoms** after downtrend.

| Phase | Key events (long side) |
|-------|------------------------|
| **A** | **PS** (preliminary support), **SC** (selling climax), **AR** (automatic rally), **ST** (secondary test) |
| **B** | Range-building; smart money accumulates |
| **C** | **Spring** / shakeout — false breakdown below support (mirror of UTAD) |
| **D** | **SOS** (sign of strength), **LPS** (last point of support) — long entry |
| **E** | Markup |

**Long entries:** Spring hold (aggressive), SOS breakout (standard), LPS retest (conservative).

**Resolution upward:** Correctly identified accumulation resolves bullish **~65–70%** (symmetric to distribution).

---

## Integrating Wyckoff into analysis

1. **Identify prior trend** — markup (look for distribution) or markdown (look for accumulation)
2. **Map the range** — BC/SC high, AR low
3. **Label current phase** — A/B/C/D/E
4. **Check volume** — effort vs result at each pivot
5. **Align with candlesticks** — UTAD = shooting star / bearish engulfing at range top; Spring = hammer at range low
6. **Project targets** — cause–effect measured move
7. **State phase in output** — e.g. *"SNDK: likely Phase C/D — UTAD rejected at $2,354"*

---

## Wyckoff vs candlestick-only analysis

| Candlestick only | + Wyckoff |
|------------------|-----------|
| "Shooting star at resistance" | "UTAD in distribution Phase C after 6-month cause" |
| Single-bar warning | Institutional exit narrative with volume map |
| Vague target | Measured move from range height |

Always present Wyckoff as **structural context**, not certainty — ranges can be re-accumulation in strong bull markets.
